# GenAI Powered Data Analytics Job Simulation

## 📊 Project Overview

This project is based on a GenAI-powered Data Analytics Job Simulation
where I applied data analysis techniques along with generative AI tools
to solve real-world business problems. The simulation focused on
extracting insights, automating analysis, and improving decision-making
using AI.

## 🛠️ Tasks Performed

-   Cleaned and preprocessed raw datasets
-   Performed exploratory data analysis (EDA)
-   Used Generative AI tools for data insights and automation
-   Built dashboards and reports
-   Generated business recommendations based on analysis

-   Business Problem Solving
